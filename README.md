# ProyectoGitDAM

Programa en Java que muestra un mensaje por consola utilizando Git y GitHub.

## Descripción

Este proyecto es una aplicación Java simple que demuestra el uso de control de versiones con Git y GitHub. El programa imprime mensajes en la consola.

## Lenguaje utilizado

- Java

## Estructura del proyecto
```
ProyectoGitDAM/
├── src/
│   └── com/
│       └── dam/
│           └── git/
│               └── HolaGit.java
└── README.md
```

## Ejecución

1. Abrir el proyecto en Eclipse
2. Navegar a la clase `HolaGit.java`
3. Ejecutar como Java Application
4. Ver el resultado en la consola

## FASE 5: Commit y Push del README

**1. Añadir el README al índice:**
- Clic derecho sobre el proyecto
- `Team → Add to Index`

**2. Hacer commit:**
- Clic derecho sobre el proyecto
- `Team → Commit...`
- Mensaje: `Añadido README.md con documentación del proyecto`
- Clic en `Commit and Push`

**3. En la ventana de Push:**
- Si ya configuraste todo antes, solo clic en `Next`
- Luego `Finish`

## FASE 6: Verificación en GitHub

**1. Ve a tu repositorio en GitHub:**
- Abre tu navegador
- Ve a `https://github.com/tu-usuario/ProyectoGitDAM`

**2. Verificar que todo está subido:**
- ✅ Debes ver la estructura de carpetas del proyecto
- ✅ El archivo `README.md` debe mostrarse renderizado en la página principal
- ✅ Deben aparecer al menos 3 commits (los 2 iniciales + el del README)

**3. Ver los commits:**
- Clic en el número de commits (arriba a la derecha)
- Verifica que aparecen todos con sus mensajes correctos
